# Examples

This directory is for **stack adapter examples** — concrete configurations showing how to set up zachflow for specific tech stacks.

v1.0 ships with the directory empty. External contributions welcome.

## What an example contains

```
examples/<your-stack>/
├── README.md                    # what stack this targets, who maintains
├── sprint-config.example.yaml   # filled-in sprint config
├── teammates/
│   ├── be-engineer.md           # filled-in BE Engineer guide for this stack
│   ├── fe-engineer.md
│   ├── design-engineer.md
│   └── evaluator.md
└── (any stack-specific scripts or skills)
```
