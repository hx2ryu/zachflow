# Contributing to zachflow

zachflow is in v0.1 (Sprint 0 bootstrap). External contributions welcome once v1.0 ships — see [`docs/roadmap.md`](docs/roadmap.md).

## Development

- This is the OSS scaffold. Production usage and stack-specific adapters are out of scope here.
- Workflow content is platform-agnostic markdown. Tool-specific calls (e.g., Claude Code `TaskCreate`) are isolated to clearly marked sections.
- Stack adapter examples (`examples/`) are external contributions — see [`examples/README.md`](examples/README.md).
